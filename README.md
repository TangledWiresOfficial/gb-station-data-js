# gb-station-data-js

This is a JS/TS library containing data from [gb-station-data](https://github.com/TangledWiresOfficial/gb-station-data). This is the library used in [Stationary](https://github.com/TangledWiresOfficial/Stationary).

## Installation
You can add this library to your project with this command:
```shell
npm add @tangledwires/gb-station-data
```
